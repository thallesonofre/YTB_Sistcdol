## Sistema de troca de óleo veícular

### Informações gerais do sistema:

1. Cadastros
   - Cliente:
     - Nome, CPF, Telefone, Email, Endereço.
   - Marca:
     - Nome.
   - Veículo:
     - Placa, Cor, Marca, Modelo, Ano, Cliente.
   - Óleo:
     - Tipo, Descrição.
   - Filtro:
     - Tipo, Descrição.
2. Lançamento
   - Troca de óleo e filtro.
3. Relatórios
   - Clientes;
   - Marcas;
   - Veículos;
   - Óleos;
   - Filtros;
